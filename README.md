# Customer Segmentation using K-Means Clustering

This project applies unsupervised machine learning (K-Means) to segment customers based on their income and spending behavior.

## Objective:
Identify distinct customer groups for better marketing strategies.

## Techniques Used:
- K-Means Clustering
- Elbow Method (Optimal clusters)
- Feature Scaling

## Tools:
Python, Pandas, Matplotlib, Scikit-learn

## Insight:
Customers are grouped into distinct segments based on spending behavior and income patterns.
